# TFM_JAAT

Código del TFM: solver de condensación en faros LED (OpenFOAM v2512).

## Estructura

```
applications/   -> Solver chtMultiRegionThinFilmFoam
src/            -> Librería humidityBoundaryConditions (BC humidityCondensation)
```

## Archivos .7z

- Están divididos en partes (.7z.001, .7z.002, ...).
- Motivo: límite de tamaño de subida de GitHub.
- Para abrir: descargar todas las partes en la misma carpeta y descomprimir la .001.
